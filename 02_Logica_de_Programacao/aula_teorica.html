1. Variáveis e Tipos de Dados

Uma variável é um espaço com nome onde você guarda um valor para usar depois. O tipo dessa variável diz que tipo de valor está guardado ali — e isso muda o que você pode fazer com ela.

TipoO que guardaExemplo
intNúmero inteiroidade = 30
floatNúmero decimalaltura = 1.75
strTexto (string)nome = "Ana"
boolVerdadeiro ou falsoativo = True
💡 Atenção: = guarda um valor (atribuição). == compara dois valores. Confundir os dois é o erro mais comum desta semana!

Exemplo 1 (resolvido) — Cadastro simples

nome = "Mariana"
idade = 28
altura = 1.68
esta_matriculada = True

print(nome, idade, altura, esta_matriculada)
print(type(nome), type(idade), type(altura), type(esta_matriculada))

Exemplo 2 (resolvido) — Trocando o tipo de um dado

Às vezes você recebe um número como texto (ex.: digitado em um formulário) e precisa converter para poder calcular com ele:

idade_texto = "25"          # chegou como texto
idade_numero = int(idade_texto)   # convertendo para número

print(idade_numero + 1)     # agora dá para somar: 26
✏️ Atividade Prática 1 (no notebook): crie 3 variáveis suas (uma int, uma float e uma str) e confirme o tipo de cada uma com type().
⚠️ Cuidado com o ... nas atividades: nas atividades práticas você vai ver linhas como minha_idade = ... — o ... é só um marcador "substitua aqui". Mas atenção: se você esquecer de trocar e executar mesmo assim, o Python não vai dar erro (o ... é um valor válido em Python, chamado Ellipsis) — o código vai rodar e mostrar algo sem sentido. Por isso, sempre leia a saída e confira se o resultado faz sentido, mesmo quando não aparece nenhuma mensagem de erro.

2. Operadores Aritméticos

Agora que você sabe guardar dados em variáveis (seção 1), o próximo passo é fazer contas com eles. É exatamente isso que os operadores aritméticos fazem.

OperadorSignificadoExemplo
+Soma5 + 2 → 7
-Subtração5 - 2 → 3
*Multiplicação5 * 2 → 10
/Divisão (sempre decimal)5 / 2 → 2.5
//Divisão inteira5 // 2 → 2
%Resto da divisão (módulo)5 % 2 → 1
**Potência5 ** 2 → 25

Exemplo 1 (resolvido) — Conta dividida entre amigos

conta_total = 150.0
numero_de_amigos = 4

valor_por_pessoa = conta_total / numero_de_amigos
print(f"Cada um paga R$ {valor_por_pessoa}")
💡 E se a conta não dividir certo? Tente trocar numero_de_amigos para 3 e rodar de novo: o resultado vira 33.333333333333336, em vez de um número redondo. Isso é normal — é assim que computadores representam números decimais, em qualquer linguagem, não é um erro seu nem do Python. Se quiser um valor mais "limpo" para mostrar, use round(valor_por_pessoa, 2) (arredonda para 2 casas decimais).

Exemplo 2 (resolvido) — Par ou ímpar com módulo

numero = 17
resto = numero % 2

print(f"O resto da divisão de {numero} por 2 é {resto}")
print("É par" if resto == 0 else "É ímpar")
✏️ Atividade Prática 2 (no notebook): dado um preco de sua escolha, calcule o preço com 10% de desconto usando os operadores aritméticos.

3. Operadores Relacionais e Lógicos

Os operadores da seção 2 produzem números (somas, descontos, restos). Os operadores desta seção são diferentes: eles produzem True ou False — e é exatamente esse True/False que vai permitir que seu programa decida algo, daqui a pouco, na seção 4.

Operadores relacionais comparam valores e sempre resultam em True ou False. Operadores lógicos combinam várias comparações.

OperadorTipoSignificado
== / !=RelacionalIgual a / diferente de
> / <RelacionalMaior que / menor que
>= / <=RelacionalMaior ou igual / menor ou igual
andLógicoVerdadeiro só se ambos os lados forem verdadeiros
orLógicoVerdadeiro se pelo menos um lado for verdadeiro
notLógicoInverte o valor (verdadeiro vira falso)

Exemplo 1 (resolvido) — Verificando maioridade

idade = 16
print(idade >= 18)   # False

Exemplo 2 (resolvido) — Combinando condições com "and"

idade = 20
tem_carteira = True

pode_dirigir = idade >= 18 and tem_carteira
print(f"Pode dirigir? {pode_dirigir}")
✏️ Atividade Prática 3 (no notebook): crie variáveis nota1 e nota2 e use and para verificar se a pessoa passou em ambas (nota ≥ 7 nas duas).

4. Estruturas Condicionais (if / elif / else)

Lembra da pergunta do início desta página, "como um programa decide sozinho se você pode ou não entrar em um site"? A resposta é: comparando um valor (seção 3) dentro de um if. A estrutura condicional faz seu programa decidir qual caminho seguir, com base numa condição que resulta em True ou False.

if condicao_1:
    # executa se condicao_1 for True
elif condicao_2:
    # executa se condicao_1 for False e condicao_2 for True
else:
    # executa se nenhuma condição anterior for True
💡 Atenção: em Python, o bloco de cada condição precisa estar indentado (4 espaços) — é a indentação que diz ao Python o que pertence a cada bloco, não chaves {}.
⚠️ Veja como é um erro real do Python: se você esquecer o : no final da linha do if, o Python recusa a executar e mostra uma mensagem assim: 
  File "<string>", line 1
    if idade >= 18
                  ^
SyntaxError: expected ':'
Isso não significa que você "quebrou" o código — é o Python te avisando exatamente onde e o que está errado. Leia sempre a última linha da mensagem primeiro: ela quase sempre diz o problema (aqui, expected ':' = "faltou o dois-pontos"). Tente provocar esse erro de propósito no notebook — é mais fácil aprender a ler um erro quando você sabe exatamente o que causou ele.

Exemplo 1 (resolvido) — Classificando uma nota

nota = 7.5

if nota >= 9:
    print("Excelente")
elif nota >= 7:
    print("Bom")
elif nota >= 5:
    print("Regular")
else:
    print("Insuficiente")

Exemplo 2 (resolvido) — Decidindo o acesso a um site

idade = 15

if idade >= 18:
    print("Acesso liberado")
else:
    print("Acesso bloqueado: conteúdo para maiores de 18 anos")

Exemplo 3 (resolvido) — Combinando "and" dentro do if

Os Exemplos 1 e 2 testavam uma condição por vez. Mas você também pode usar os operadores lógicos da seção 3 (and, or) dentro da condição do if, para testar várias coisas ao mesmo tempo:

numero = 15

if numero % 3 == 0 and numero % 5 == 0:
    print(f"{numero} é múltiplo de 3 e de 5 ao mesmo tempo")
elif numero % 3 == 0:
    print(f"{numero} é múltiplo de 3")
elif numero % 5 == 0:
    print(f"{numero} é múltiplo de 5")
else:
    print(f"{numero} não é múltiplo de 3 nem de 5")
💡 Por que o and vem primeiro? Se você checasse "múltiplo de 3" antes do "múltiplo de 3 e de 5", o programa pararia no primeiro elif verdadeiro e nunca chegaria a avisar que o número é múltiplo dos dois. Ordem importa: sempre comece pela condição mais específica.

Exemplo 4 (resolvido) — Calculando e depois classificando

Até aqui, toda condição comparava uma variável já pronta (idade, nota). Mas você também pode calcular um valor novo (com os operadores da seção 2) e só depois classificá-lo com if/elif/else:

renda_mensal = 3000
gasto_planejado = 900

razao = gasto_planejado / renda_mensal   # calcula primeiro...

if razao <= 0.30:                        # ...e só depois classifica
    print("Gasto dentro do limite recomendado (até 30% da renda)")
elif razao <= 0.50:
    print("Gasto no limite de atenção (30% a 50% da renda)")
else:
    print("Gasto acima do recomendado (mais de 50% da renda)")
✏️ Atividade Prática 4 (no notebook): dada uma idade, use if/elif/else para classificar em "Criança" (até 12), "Adolescente" (13–17) ou "Adulto" (18+).

5. Estruturas de Repetição (for / while)

Com if/elif/else (seção 4) seu programa já decide um caminho — mas só uma vez. Agora você vai aprender a repetir uma ação várias vezes, sem copiar e colar o mesmo código.

Use for quando você já sabe quantas vezes quer repetir (ex.: "para cada número de 1 a 10"). Use while quando a repetição depende de uma condição que pode mudar (ex.: "enquanto o saldo for maior que zero").

Exemplo 1 (resolvido) — Contando com for

for numero in range(1, 6):
    print(f"Contando: {numero}")

Exemplo 2 (resolvido) — Contagem regressiva com while

contador = 5

while contador > 0:
    print(contador)
    contador = contador - 1

print("Lançamento!")
✏️ Atividade Prática 5 (no notebook): use for para percorrer os números de 1 a 10 e imprimir apenas os números pares (use o operador % que você já conhece).
